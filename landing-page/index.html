How It Works

A robust, 5-step LangGraph state machine powers the backend.

01

Ingest

Fetch Data Quality failures from the OpenMetadata API.

02

Govern

Extract table-level governance tags (PII, Tier1).

03

Trace

Calculate the downstream lineage blast radius.

04

Synthesize

LLM formulates a root-cause hypothesis.

05

Dispatch

Deliver payload to Slack, Sheets, or MCP client.

Install via MCP

Add this configuration to your claude_desktop_config.json to instantly enable DQ-Agent.

{
  "mcpServers": {
    "DQ-Agent": {
      "command": "/absolute/path/to/venv/bin/python",
      "args": [
        "/absolute/path/to/dq-agent/mcp_server.py"
      ],
      "env": {
        "GEMINI_API_KEY": "your_gemini_key_here",
        "SLACK_BOT_TOKEN": "your_slack_token_here",
        "SHEET_ID": "your_google_sheet_id_here"
      }
    }
  }
}

Run Manually (CLI)

Don't want to use MCP? You can trigger reports manually from your terminal.

# Run a full check against your live OpenMetadata instance
python agent.py

# Run an offline test with mock data (no live OM needed)
python agent.py --mock

# Filter the report for a specific domain
python agent.py --domain finance
